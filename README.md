# graphql-relay-todo

에디켓 스택에 적응하기 위한 TODO MVC 개발.

## 요구사항
- TODO 만들기
- TODO 지우기

## 사용해야하는 스택들
- React
- (Material-ui)
- React-Router 4
- Test (jest)
- Graphql
- Graphql-relay
- Webpack
- Express
- CSS Module + PostCSS/
- (WebpackDevServer)
- Babel
